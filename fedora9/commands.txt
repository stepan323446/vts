7 Март:
==========================
---- Команды ----:
ls - показывает расположение
rm - удалить файл
cd - Перейти в директорию
mkdir - создать директорию
rmdir - удалить директорию
pwd - текущая директория
cat - Просмотр/Создание файла
cp - Копировать
mv - Переместить

--- Теория ----
Нельзя ставить специальные символы в названии файлы
Максимальное количество символов: 256
cd . - Текущая директория
cd ../ - Родительская директория
cd ~ - начальная (пользовательская) директория
cd / - корневая (korenski) dir

3 типа:
- običan
d dir/folder
l - link

cat > [Название файла]
Content
Ctrl + D - сохранить. Новый файл с контентом

cat >> [Название файла]
Content
Ctrl + D - добавить строку

cat file1 >> file2 - Копировать файл (как cp)
